# MediCore EMR

Electronic Medical Records System built with Spring Boot.

## Features

- JWT Authentication
- Role-Based Access Control
- Patient Management
- Medical Records
- Kafka Event Streaming
- RabbitMQ Messaging
- Swagger Documentation

## Tech Stack

- Java 21
- Spring Boot 3
- PostgreSQL
- Kafka
- RabbitMQ
- Docker

## Running Locally

mvn spring-boot:run
